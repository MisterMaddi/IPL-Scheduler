# 🏏 IPL Scheduling System (Raylib GUI)

A full-featured IPL tournament scheduling system built in C with a graphical interface using Raylib. The application automates match scheduling while handling real-world constraints like venue availability, player fatigue, and time-slot optimization.

---

## 🚀 Overview

This project simulates an IPL-style cricket tournament using a round-robin scheduling algorithm. It ensures fair match distribution, avoids consecutive matches, and dynamically updates the points table and playoffs.

---

## ✨ Features

* 📅 Automated match scheduling (round-robin format)
* 🏟️ Venue and restriction handling
* ⛔ Avoids consecutive matches (player fatigue logic)
* 🗓️ Weekend double-header scheduling
* 📊 Points table with Net Run Rate (NRR)
* 🌧️ Match cancellation simulation (rain)
* 🏆 Playoff system (Qualifier 1, Eliminator, Final)
* 🖥️ Interactive GUI using Raylib

## ▶️ How to Run

### 1. Install Raylib (Mac)

```bash
brew install raylib
```

### 2. Compile

```bash
gcc main.c -o ipl -lraylib -framework OpenGL -framework Cocoa -framework IOKit -framework CoreVideo
```

### 3. Run

```bash
./ipl
```
